# Support

Use these channels depending on what you need.

## Questions

Open a GitHub Discussion when available, or create an issue using the question template.

Include:

- Laravel version.
- PHP version.
- ModelMind version.
- Installation method.
- Relevant config with secrets removed.

## Bugs

Use the bug report template and include a minimal reproduction when possible.

Good reports include:

- Expected behavior.
- Actual behavior.
- Steps to reproduce.
- Relevant stack trace.
- Whether published views were customized.
- Whether the issue appears in light theme, dark theme, or both.

## Security

Do not create a public support issue for security problems. Follow [SECURITY.md](SECURITY.md).

## What Not to Share

Never post:

- OpenAI API keys.
- `.env` files.
- Passwords or tokens.
- Customer data.
- Full production database records.
- Private business data used as model context.
