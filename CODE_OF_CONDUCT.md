# Code of Conduct

## Our Pledge

We are committed to making ModelMind a welcoming, respectful, and productive project for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, but take responsibility for impact.
- Keep discussions focused on improving the package.
- Give clear technical context when reporting problems.
- Respect maintainers' time and project boundaries.

## Unacceptable Behavior

- Harassment, insults, threats, or personal attacks.
- Discriminatory language or conduct.
- Publishing private information without permission.
- Spam, trolling, or repeated off-topic comments.
- Sharing secrets, API keys, credentials, or sensitive user data in public issues.

## Enforcement

Maintainers may edit, hide, lock, or delete comments and issues that violate this code. Maintainers may also temporarily or permanently block contributors who repeatedly violate it.

## Reporting

Report conduct concerns privately through GitHub repository owner contact channels. Security issues should follow [SECURITY.md](SECURITY.md).
